Basit bir borsa oyunu örneği (konsol uygulaması).

Kullanım:

- Projeyi çalıştırmak için:

```bash
dart run
```

- Komutlar (oyun içinde):
	- `l` : Piyasadaki hisseleri ve fiyatlarını listeler
	- `p` : Portföyünüzdeki nakit ve hisseleri gösterir
	- `b SYMBOL QTY` : Belirtilen hisseden `QTY` adet alır
	- `s SYMBOL QTY` : Belirtilen hisseden `QTY` adet satar
	- `n` : Bir gün ilerletir (fiyatlar rastgele değişir)
	- `h` : Yardım
	- `q` : Çıkış

Projede `lib/borsa_oyunu.dart` temel oyun mantığını içerir.
